HTML

HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It provides the basic framework for web content by defining elements such as text, images, links, and multimedia.

CSS

Cascading Style Sheets (CSS) courses cover the basics of CSS, how to apply CSS to HTML, and how to create responsive layouts. CSS courses may also include advanced techniques like animations and transitions

FIGMA

Figma is a tool for creating interfaces and graphics collaboratively. Sign up, explore the interface, and use the toolbar for shapes and text. Learn to create shapes, adjust them with tools like Move and Scale, and organize layers.
